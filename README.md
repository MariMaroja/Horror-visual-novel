# Echoes in the Dark

A psychological horror visual novel built with HTML, CSS and JavaScript.

Explore an abandoned house, uncover disturbing secrets hidden within forgotten letters, and make choices that determine your fate. Every decision matters, and some doors are better left unopened.

---

## Overview

Echoes in the Dark is a browser-based horror experience focused on atmosphere, exploration, and narrative choices.

Players navigate through a mysterious house, collect clues, discover hidden items, and unlock multiple endings while uncovering the truth behind the infamous Red Door.

This project was developed to practice frontend development, state management, DOM manipulation, and interactive storytelling using vanilla JavaScript.
